PlusPGP

Google+ users PGP keys collection

Download as .zip Download as .tar.gz View on GitHub

Here is a list of PGP public keys currently collected. To submit your public key, mention me in a private post on Google+, which contains the following information:

  • Your pub key ID
  • Corresponding Email address
  • PGP server information
When noticed I will send you an ENCRYPTED Email with my personal signature, and you have to send the content back to me, to verify you are the owner of this key.

To import all of them, do 

$ git clone https://github.com/phoenixlzx/PlusPGP.git
$ cd PlusPGP/
$ gpg --import *.asc  # Or specify the filename.
